Isomorphisms

    Help Guide

    At the top, the user can select a T-Norm, which is crucial for computing fuzzy twin width and fuzzy similarity.

    The user can set the membership function values (0-1) for nodes and edges before adding them.

    • "Add Node" adds nodes to the drawing area.
    • To add an edge between two nodes:
      • Click on the nodes you want to connect (they will turn green).
      • Click "Add Edge" to create the connection.
    • To delete an object (node or edge), right-click on it.

    The "Compute" button calculates fuzzy graph similarity and twin width.

    • If the graphs are isomorphic, the "Show Isomorphisms" button will be enabled.
    • This button allows you to view all possible graph mappings.
