Isomorphisms

    Help Guide

    Select T-Norm sets T-Norm crucial for creating a fuzzy graph, computing Fuzzy Twin Width and Fuzzy Graph Similarity.

    Node Degree defines the membership function value of a newly added node.

    Edge Degree defines the membership function value of a newly added edge.

    Add Node adds node to the drawing area.

    Add Edge adds edge between selected nodes to the drawing area.

    • To add an edge between two nodes:
      • Click on the nodes you want to connect (they will turn green).
      • Click Add Edge to create the connection.

    Delete Graph deletes whole fuzzy graph.

    • To delete an object (node or edge), right-click on it.

    Compute calculates Fuzzy Graph similarity and Fuzzy Twin Width.

    • The error value X will be displayed if:
      • Fuzzy Twin Width: the fuzzy graph is missing
      • Fuzzy Graph Similarity: the fuzzy graphs are not isomorphic.
    • If the graphs are isomorphic, the Show Isomorphisms button will be enabled.
      • This button allows you to view all possible graph mappings.
